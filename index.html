1.Arithmetic Operators

Arithmetic operator takes numerical values (either lliterals or variables) as their operands and returns a single numerical value.

Arithmetic Operators types :

  1. Addition (+)
  2. Subtraction (-)
  3. Multiplication (*)
  4. Division (/)
  5. Modulus (%)
  6. Exponentiation (**)
  7. Increment (++)
  8. Decrement (--)

A. Addition (+)

Example :

let a = 10;
let b = 20;
let c = a + b;
console.log(c);

B. Subtraction (-)

Example :

let d = 12;
let e = 10;
let f = e - d;
console.log(f);

C. Multiplication (*)

Example :

let g = 10;
let h = 2;
let i = g * h;
console.log(i);

D. Division (/)

Example :

let j = 100;
let k = 5;
let l = j / k;
console.log(l);

E. Modulus (%)

Example :

let m = 20;
let n = 2;
let o = m % 2;
console.log(o);

F. Exponentiation (**)

Example :

let p = 12;
let q = 2;
let r = p ** q;
console.log(r);

G. Increment (++)

  1. Pre-increment (++x)
  2. Post-increment (x++)

Example : 1

let s = 12;
let t = ++s;
console.log(t);

Example : 2

let u = 15;
let v = u++;
console.log(v);

H. Decrement (--)

  1. Pre-decrement (--x)
  2. Post-decrement (x--)

Example : 1

let w = 10;
let x = --w;
console.log(x);

Example : 2

let y = 20;
let z = y--;
console.log(z);

3. Comparision operators

The comparison operators in JavaScript are used to compare two values and return a boolean value (true or false) depending on the result of the comparison.

Comparision operators types :

  1. Equal to (==)
  2. Not equal to (!=)
  3. Strict equal to (===)
  4. Strict not equal to (!==)
  5. Greater than (>)
  6. Less than (<)
  7. Greater than or equal to (>=)
  8. Less than or equal to (<=)

A. Equal to (==)

Example :

var aa = 10;
var bb = 10;
console.log(aa == bb);

B. Not equal to (!=)

Example :

var aa = 10;
var bb = 20;
console.log(aa != bb);

C. Strict equal to (===)

Example :

var aa = 10;
var bb = "20";
console.log(aa === bb);

D. Strict not equal to (!==)

Example :

var aa = 10;
var bb = "20";
console.log(aa !== bb);

E. Greater than (>)

Example :

var aa = 20;
var bb = 10;
console.log(aa > bb);

F. Less than (<)

Example :

var aa = 10;
var bb = 20;
console.log(aa < bb);

G. Greater than or equal to (>=)

Example :

var aa = 10;
var bb = 10;
console.log(aa >= bb);

H. Less than or equal to (<=)

Example :

var aa = 10;
var bb = 10;
console.log(aa <= bb);

4. Logical Operators

Logical operators are typically used with Boolean (logical) values; when they are, they return a Boolean value.

Logical operators types :

  1. AND (&&)
  2. OR (||)
  3. NOT (!)

A. AND (&&)

Example :

var xx = 5;
var yy = 10;
if (xx < 10 && yy > 5) {
console.log("Both conditions are true");
} else {
console.log("At least one condition is false");
}

B. OR (||)

Example :

var xx = 5;
var yy = 10;
if (xx < 10 || yy < 5) {
console.log("At least one condition is true");
} else {
console.log("Both conditions are false");
}

C. NOT (!)

Example :

var xx = 5;
var yy = 10;
if (!(xx > 10)) {
console.log("x is not greater than 10");
}
if (!(yy < 5)) {
console.log("y is not less than 5");
}

5. Conditional (Ternary) Operator

The conditional operator is the only JavaScript operator that takes three operands. The operator can have one of two values based on a condition.

Conditional (Ternary) Operator

Example :

var xx = 5;
var result = xx > 10 ? "xx is greater than 10" : "xx is not greater than 10";
console.log(result);

6. Bitwise Operators

A bitwise operator treats their operands as a set of 32 bits (zeros and ones), rather than as decimal, hexadecimal, or octal numbers. For example, the decimal number nine has a binary representation of 1001.

Bitwise Operators Types :

  1. AND operator (&)
  2. OR operator (|)
  3. XOR operator (^)
  4. NOT operator (~)
  5. Left shift operator (<<)
  6. Right Shift operator (>>)
  7. Unsigned Right Shift operator (>>>)

A. AND operators (&)

Example :

var aa = 10; var bb = 20; var result = aa & bb; console.log(result);

B. OR operators (|)

Example :

var aa = 10; var bb = 20; var result = aa | bb; console.log(result);

C. XOR (^)

Example :

var aa = 10; var bb = 20; var result = aa ^ bb; console.log(result);

D. NOT

Example :

var aa = 10; var bb = -16; console.log(~aa); console.log(~bb);

E. Left shift

Example :

var aa = 8; var bb = 3; var result = aa << bb; console.log(result);

F. Right shift operator (>>)

Example :

var aa = 5; var bb = 2; var result = aa >> bb; console.log(result);

G. Unsigned Right shift operator (>>>)

Example :

var aa = 8; var bb = 2; var result = aa >>> bb; console.log(result);

7. String Operators

In addition to the comparison operators, which can be used on string values, the concatenation operator (+) concatenates two string values together, returning another string that is the union of the two operand strings.

Example :

console.log("Hello" + "World");

8. Comma Operators

The comma operator (,) evaluates both of its operands and returns the value of the last operand. This operator is primarily used inside a for loop, to allow multiple variables to be updated each time through the loop.

Example :

var aa = 10, bb = 20;
var result = (aa++, bb++, aa + bb);
console.log(aa);
console.log(bb);
console.log(result);

11. Unary Operator

In JavaScript, unary operators are operators that operate on a single operand.

Unary operators types :

  1. Plus (+)
  2. Negative (-)
  3. Logical NOT (!)
  4. Bitwise NOT (~)
  5. typeof
  6. void
  7. delete

Plus (+)

Example :

var aa = 10; var bb = "10"; var result = +aa; var result1 = +bb; console.log(result); console.log(result1);

Negative (-)

Example :

var aa = 10; var bb = "10"; var result = -aa; var result1 = -bb; console.log(result); console.log(result1);

Logical NOT (!)

Example :

var aa = 10; var bb = 15; var result = !(aa > bb); console.log(result);

Bitwise NOT (~)

Example :

var aa = 10; var result = ~aa; console.log(result);

typeof
